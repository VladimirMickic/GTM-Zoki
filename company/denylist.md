# Discover denylist

Domains `discover()` must never emit as prospects — resellers, dealers, marketplaces,
listicle farms. Maintained by the learn loop: each entry cites the run that caught it.
Format: `- <domain> — <reason> (<run>, <date>)`.

- advexure.com — multi-brand drone reseller/dealer (discover-1, 2026-07-18)
- enterprise.dronenerds.com — enterprise drone dealer, sells other brands (discover-1, 2026-07-18)
- dronenerds.com — same dealer, consumer storefront (discover-1, 2026-07-18)
- ledrones.org — law-enforcement drone shop, sells other makers' bundles (discover-1, 2026-07-18)
- unmannedsystemstechnology.com — trade-news site, not a maker (discover-2, 2026-07-18)
- abjacademy.global — drone blog/listicle site (discover-3, 2026-07-18)
